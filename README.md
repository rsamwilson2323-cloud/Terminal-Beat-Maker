# 🎵 Terminal Beat Maker

Terminal Beat Maker is a fun keyboard-controlled beat machine built using **Python 🐍** and Windows sound libraries.

The project includes two versions:

🎧 **Terminal Beat Maker** – Simple A–Z keyboard beat generator.

🌈 **Colorful Beat Maker** – Enhanced version with colorful terminal effects, beat names, pulse animations, and multi-tone sound patterns.

Press any letter from A–Z to play unique beats and create your own rhythm directly from the keyboard.

---

# ✨ Features

🎹 A–Z Keyboard Beat Control

* 26 unique beats

🎧 Real-Time Sound Generation

# 🧠 How It Works

🎵 Frequency Mapping

Each letter is assigned a unique frequency.

🎹 Keyboard Detection

Reads key presses in real time.

🔊 Sound Playback

Generates tones using Windows Beep functions.

🌈 Visual Effects

Displays colorful pulse animations and beat names.
